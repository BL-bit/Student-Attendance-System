## 项目简介
本项目为嘉应学院计算机学院软件项目管理课程作业，实现面向师生的学生考勤管理功能，包含签到、请假、考勤统计、权限管理模块。

## 一级目录功能说明
1. backend：后端服务代码，实现签到、请假、考勤统计、用户权限业务接口
2. frontend：前端页面，分为学生端、教师端、管理员端三大模块
3. sql：数据库初始化脚本，包含学生、课程、考勤记录、请假申请数据表
4. docs：软件项目管理配套文档，包含RAM责任分配矩阵、小组分工记录表
5. tools：辅助工具脚本，支持批量导入学生信息、导出考勤Excel报表
